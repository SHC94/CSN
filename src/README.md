## 로그
**출력 형식**
- 클래스명::메서드명::메세지 ex) MemberRepository::save::"회원저장완료"

**로그 레벨**

1. 운영 방식
- 운영 및 테스트로 구별하여 로그를 출력한다.
- 운영에서는 error 레벨만을 출력한다. 
- 테스트 및 개발 환경에서는 info 이상을 출력한다. 

2. info
- 테스트할 때 필요한 출력에 사용한다. 

4. error
- 예외가 발생했을 때 사용한다.
